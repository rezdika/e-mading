Test Form Komentar Sederhana




Instruksi Testing:

  1. Login sebagai siswa di aplikasi utama
  2. Buka halaman artikel (blog-details)
  3. Buka Developer Tools (F12)
  4. Pergi ke tab Console
  5. Coba submit komentar
  6. Lihat apakah ada error di console
  7. Pergi ke tab Network, lihat request yang dikirim

Yang Harus Diperiksa:

  • Apakah request POST terkirim ke /articles/{id}/comments?
  • Apakah response code 200, 302, atau error?
  • Apakah ada error JavaScript di console?
  • Apakah CSRF token valid?
  • Apakah user benar-benar authenticated?
